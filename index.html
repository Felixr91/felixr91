About

Felix Ramirez is a full stack web developer with experience building web applications utilizing RESTful architecture. He is a quick learner, able to adapt to changes in software technology and programming languages.

Felix specializes in Front End Development. He brings his passion for creating beautifully responsive applications to every project.

In the past, Felix has been a radio board operator at Sirius XM Satellite Radio, and a High School Special Education Science Teacher in New York.

Felix also enjoys music production, jazz, guitar, photography, typography, cinematography, and graphic design.

"Holla!"

The "Holla!" kanban project was collaboratively built with friend and colleague, Malik Whitfield. The application utilizes Vue.js on the client side and Node.js on the server side. In Japanese, the word "Kan" means "visual" and "ban" means "card," so Kanban refers to visual cards. Click the link to register and test the app!
